编译准备#

编译系统： ubuntu 软件包： apt-get install build-essential subversion libncurses5-dev zlib1g-dev gawk gcc-multilib flex git-core get text 源码：git clone git://git.openwrt.org/14.07/openwrt.git

开始编译#

准备用户： useradd -d /home/xxx -m xxx 更新源：

./scripts/feeds update -a
./scripts/feeds install -a

编译配置：make menuconfig

编译： make V=s
