使用Launchrocket管理MacOS的服务

April 3, 2018 · 1 min · Me | Suggest Changes

经常使用MacOS做web开发的同学们，当我们使用brew安装nginx、php后，管理其守护进程的生命周期就基本只能靠类似brew services start nginx管理了，要是有一个GUI程序为我们管理这些服务进程就好了。

此处应该有Windows下的WAMP的截图，稍后补！

项目地址jimbojsb/launchrocket

# 执行命令前需要安装brew
brew cask install launchrocket

安装之后就可以移步系统设置页面，找到launchrocket

老高目前只安装了nginx，通过界面我们可以看到我们可以选择启动，结束，自动启动和移除。

如果没有显示出你的服务，点击一下Scan Homebrew即可。

从此可以告别命令行管理服务了，是不是爽歪歪！

补充卸载命令：

brew cask uninstall launchrocket
