File-already-exists--filesystem-解决办法

今天用svn命令行提交版本的时候，碰到了这个比较麻烦的问题 svn: File already exists: filesystem 'xxx/svn/xxx/db' 搜了一下解决办法，都是需要两次commit，太麻烦。 直接在提交根目录执行以下命令 svn update path/ --accept=mine-full 一句话解决！
