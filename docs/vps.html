如何升级套餐#

1. [点此登陆帐户][39]
2. 在页面最下面找到 -> My Services menu
3. 点击 Manage
4. 找到 Management Actions -> Upgrade/Downgrade
5. 选择合适的套餐升级，补差价
6. 瞬间升级完毕
