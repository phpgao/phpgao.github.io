功能#

废话！就是显示文章目录。

2015-05-18更新：

使用simple_html_dom，放弃了正则。

安装#

命令行#

进入typecho的usr/plugins目录

使用git clone https://github.com/phpgao/TableOfContents.git

这样plugins下应该会有TableOfContents文件夹，然后在后台->插件处安装。

FTP#

解压下载的文件，将TableOfContents上传至usr/plugins目录即可！
